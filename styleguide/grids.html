Flexbox grid

Below are examples of implementing scalable, responsive, flexbox grids. A familiar 12 column grid system is used, and there are both automatic and manual styling options.

Markup-only (automatic) grids

Automatic grids allow you to simply create HTML markup with some specific attributes to get a scalable, responsive grid working quickly. These are useful for grids of equal-sized columns, grids with gutters, or a quick start to a more complex grid.

Here is an example of a simple automatic grid with three columns of equal widths and a gutter: 

                    
                        <div role="grid" sf-columns="3" sf-gutters>
                            <div role="gridcell"></div>
                            <div role="gridcell"></div>
                            <div role="gridcell"></div>
                        </div>
FUN FACT: SFUMATO grid cell gutters (horizontal) can be rems or percentages that are used for right margins. Grid cell widths have the gutter subtracted using calc(). The grid's right edge uses a negative margin to negate the extra space after the last grid cell in a row. Grid cell bottom gutters are margin values in rems. The grid container will use a negative bottom gutter value so that the last row of items doesn't push subsequent tags down. This is all triggered by the use of the sf-gutters attribute.

Grid attributes...

Flexbox grids have options to control their behavior, saving you the trouble of writing SCSS code.

  • sf-columns="#" — values: 1-12
    This tells the grid how many equal-width columns to assume as a default. You can always change the grid cell widths and other styles in your own SCSS code. A value of 1 may seem silly, but on full-width grid cells it helps SFUMATO handle the margins.
  • sf-reverse
    This makes the grid cells render in reverse order when they are not stacked (e.g. mobile). If reverse is desired for stacked grid cells, simply assign flex-direction: column-reverse; to the grid for the desired breakpoint.
  • sf-equal-heights
    All grid cells in a row will be the height of the tallest grid cell.
  • sf-centered
    All grid cells in a row will vertically center their direct descendants.
  • sf-gutters
    This tells SFUMATO to render grid cells with gutters on the right and bottom. The gutter measurements can be changed in the _config-measurements.scss file.

Markup and SCSS (manual) grids

If you need full control of your grid layout, you'll want to use this type of implementation. Here, the basics of grid behavior are handled for you, but you control things like grid cell widths, gutters, etc.

Here is an example of a simple manual grid with three columns of varied widths. Two have vertically centered content, and one has a right gutter. This example is the SFUMATO logo and version bar at the top of the page:

HTML markup

                            
                                <header>
                                    <div role="grid" sf-centered>
                                        <div role="gridcell">
                                            <img src="images/SFUMATO-mark-color.svg" />
                                        </div>
                                        <div role="gridcell">
                                            <h1>SFUMATO</h1>
                                        </div>
                                        <div role="gridcell">
                                            <div>
                                                <strong>Version</strong> 1.0.0
                                            </div>
                                        </div>
                                    </div>
                                </header>

SCSS markup

                            
                                header {

                                    [role="grid"] {

                                        > [role="gridcell"] {
                        
                                            &:nth-child(1) {
                        
                                                width: 15%;
                                                margin-right: 1%;
                        
                                                @include sf-media($from: tablet-portrait) {
                                                 
                                                    width: 5%;
                                                }
                                            }
                        
                                            &:nth-child(2) {
                        
                                                width: 84%;
                        
                                                @include sf-media($from: tablet-portrait) {
                        
                                                    width: 64%;
                                                }
                        
                                                h1 {
                        
                                                    margin: 0;
                                                    padding: 0;
                                                    line-height: 0.83;
                                                    font-weight: 400;
                                                    color: sf-get-color(slate);
                                                    font-family: $font-logo;
                                                    text-align: right;
                                                    font-size: 3.65rem;
                        
                                                    @include sf-media($from: tablet-portrait) {
                        
                                                        font-size: 4.5rem;
                                                        line-height: 0.75;
                                                        text-align: left;
                                                    }
                                                }
                                            }
                        
                                            &:nth-child(3) {
                        
                                                width: 100%;
                                                margin-top: 0.5rem;
                                                font-size: 1.25rem;
                                                text-align: center;
                        
                                                @include sf-media($from: tablet-portrait) {
                        
                                                    width: 30%;
                                                    margin-top: 0;
                                                    text-align: right;
                                                }
                                            }
                                        }
                                    }
                                }

SCSS markup

                            
                                #grid-example {

                                    > [role="gridcell"] {
        
                                        @include sf-set-width (
                                            $from: tablet-portrait,
                                            $width: one-third
                                        );
        
                                        &:nth-child(3), &:nth-child(4), &:nth-child(5) {
        
                                            @include sf-set-width (
                                                $from: tablet-portrait,
                                                $width: one-sixth
                                            );
                                        }
                                    }
                                }
